Accessibility Statement

Our commitment to making TradesmanFinance.co.uk accessible to everyone

Our Commitment to Accessibility

Tradesman Finance is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to achieve these goals.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

TradesmanFinance.co.uk is partially conformant with WCAG 2.1 level AA. "Partially conformant" means that some parts of the content do not fully conform to the accessibility standard.

What We've Implemented

Keyboard Navigation

  • All interactive elements can be accessed using only a keyboard
  • Visible focus indicators help you see which element has focus
  • Skip links allow quick navigation to main content
  • Logical tab order throughout all pages

Screen Reader Support

  • Proper heading hierarchy (H1-H6) for content structure
  • Descriptive link text and button labels
  • ARIA labels and landmarks for enhanced navigation
  • Alternative text for images and icons
  • Live regions for dynamic content updates

Visual Design

  • High contrast color schemes meeting WCAG AA standards (4.5:1 ratio for normal text, 3:1 for large text)
  • Text can be resized up to 200% without horizontal scrolling
  • Consistent navigation and layout throughout the site
  • Support for high contrast mode

Forms and Calculator

  • All form fields have associated labels
  • Error messages are clearly identified and announced to screen readers
  • The loan calculator is fully keyboard accessible with range slider support
  • Input validation with real-time feedback

Mobile Accessibility

  • Touch targets are at least 44x44 pixels for easy interaction
  • Content reflows appropriately on different screen sizes
  • Pinch-to-zoom is supported and not disabled

Accessibility Features

Keyboard Shortcuts

Tab
Move to next interactive element
Shift + Tab
Move to previous interactive element
Enter or Space
Activate buttons and links
Arrow Keys
Navigate within menus and sliders
Escape
Close modal dialogs or menus

Calculator Accessibility

Our loan calculator includes enhanced accessibility features:

  • Keyboard navigation: Use Tab to move between controls, arrow keys to adjust sliders
  • Screen reader announcements: Changes to loan calculations are announced
  • Input validation: Clear error messages when values are outside acceptable ranges
  • Help text: Descriptive text explains each control

Browser and Assistive Technology Support

This website has been tested with the following combinations:

Desktop

  • NVDA with Firefox and Chrome
  • JAWS with Internet Explorer, Firefox, and Chrome
  • VoiceOver with Safari
  • Dragon NaturallySpeaking with Internet Explorer, Firefox, and Chrome

Mobile

  • VoiceOver with Safari on iOS
  • TalkBack with Chrome on Android

Known Issues

We are aware of the following accessibility issues and are working to address them:

  • Some third-party embedded content may not meet accessibility standards
  • Complex animations may cause issues for users with vestibular disorders (reduced motion support is implemented)

Formal Complaints

Technical Specifications

Accessibility of TradesmanFinance.co.uk relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML5
  • CSS3
  • JavaScript (with graceful degradation)
  • WAI-ARIA

Assessment Approach

Tradesman Finance assessed the accessibility of TradesmanFinance.co.uk through:

  • Self-evaluation using automated testing tools
  • Manual testing with keyboard navigation
  • Screen reader testing with NVDA and VoiceOver
  • Color contrast analysis
  • Mobile accessibility testing
